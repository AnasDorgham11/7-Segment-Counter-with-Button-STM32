# 7-Segment-Counter-with-Button-STM32
An embedded systems project that increments a single-digit number (0 → 9) on a 7-segment display using a push button.   When the display reaches 9 and the button is pressed again, it resets back to 0.
---

## 📌 Project Overview

This project demonstrates:

- Interfacing a 7-segment display with STM32
- Configuring GPIO pins as outputs (for segments)
- Configuring a GPIO pin as input (for push button)
- Implementing a counter from 0 to 9
- Resetting the counter to 0 after 9
- Basic button debouncing
